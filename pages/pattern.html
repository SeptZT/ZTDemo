正则表达式是一个描述字符模式的对象

模式修饰符参数

i  忽略大小写
var pattern = /box/g;
var pattern = new RegExp('box', 'g');
g  全局匹配
m  多行匹配

测试正则的方法

1、test()
pattern.test(str); //返回true， false
2、exec()
pattern.exec(str); //返回结果数组， 执行失败返回null

var pattern = /Box/i;

var pattern2 = /Box/;

var str ='This is a box!';

console.log("pattern结果数组：",pattern.exec(str)); //pattern结果数组： ["box", index: 10, input: "This is a box!"]

console.log("pattern2结果数组：", pattern2.exec(str)); //pattern2结果数组： null

字符串的正则表达式方法（String对象的方法）

1、match(pattern)
返回字符串中的字串或null(默认只查找一个，设置g全局)
2、replace(pattern, replacement)
3、search(pattern)
返回字符串中pattern开始的位置或返回-1
4、split(pattern)
返回字符串按指定pattern拆分的数组
